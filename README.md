# 🎉 Simple Anniversary

> 一个温馨的纪念日倒计时应用，帮助你记住每一个重要的日子

## 💚 关于这个项目

Simple Anniversary 是一个简洁而温馨的网页应用，采用清新的绿色主题设计。它能帮助你记录和追踪生活中的重要纪念日——无论是生日、结婚纪念日、工作纪念日，还是其他对你来说特别的日子。

这个应用会为你：
- 📅 记录所有重要的纪念日
- ⏰ 实时显示距离每个日子还有多少天
- 🔔 在纪念日临近时友好地提醒你
- 💾 在你的浏览器本地安全保存数据

## ✨ 主要特性

### 🎨 清新的绿色主题
精心设计的薄荷绿配色方案，让使用过程更加舒适愉悦。

### ⚡ 实时倒计时
自动计算并显示距离每个纪念日还有多少天，让你不会错过任何重要时刻。

### 🏷️ 分类管理
支持多种类别（生日、结婚纪念日、工作纪念日等），让你的纪念日井井有条。

### 🔔 智能提醒
可以设置在纪念日前 1 天、3 天或 7 天收到浏览器通知提醒。

### 📱 响应式设计
无论是在电脑、平板还是手机上，都能完美显示和使用。

### 🔐 隐私保护
所有数据都存储在你的浏览器本地，无需注册账号，完全保护你的隐私。

## 🛠️ 技术栈

这个应用使用现代化的 Web 技术构建：

- **React 19** - 最新的 React 框架
- **Vite 7** - 极速的前端构建工具
- **date-fns** - 优雅的日期处理库
- **localStorage** - 浏览器本地存储
- **Notification API** - 浏览器原生通知功能

## 📖 使用指南

### 添加纪念日

1. 点击页面顶部的 **"+ Add Anniversary"** 按钮
2. 填写纪念日的相关信息：
   - 标题（比如"妈妈的生日"）
   - 日期
   - 描述（可选，记录一些特别的回忆）
   - 类别（生日、结婚纪念日、工作纪念日等）
3. 设置是否需要提醒，以及提前多少天提醒
4. 点击 **"Save"** 保存

### 编辑纪念日

在每个纪念日卡片上，点击 **"Edit"** 按钮即可修改信息。

### 删除纪念日

点击 **"Delete"** 按钮，确认后即可删除。

### 启用通知提醒

第一次添加纪念日后，应用会请求你允许发送通知。点击 **"Enable"** 按钮授权后，你就能在纪念日临近时收到温馨的提醒了。

## 💡 开发说明

### 设计理念

这个应用的设计遵循以下原则：
- **简洁优先**：功能直观，操作简单
- **视觉舒适**：清新的绿色主题，不会产生视觉疲劳
- **注重隐私**：所有数据本地存储，不上传服务器
- **贴心提醒**：智能通知，让你不会忘记重要的日子

### 核心功能

1. **数据持久化**：使用 localStorage 在浏览器本地保存数据
2. **实时更新**：倒计时会自动在每天零点更新
3. **智能排序**：纪念日按照时间远近自动排序，最近的排在前面
4. **视觉提示**：临近的纪念日（7天内）会有特殊的视觉标记
